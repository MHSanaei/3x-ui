只开放你实际会用到的端口。下面列出了常用端口，并提供了一个用于生成
`ufw` 和 `nftables` 规则的生成器。

## 常用端口

| Port (default) | Purpose                                   |
| -------------- | ----------------------------------------- |
| `22`           | SSH（务必保持开放！）。                    |
| `2053`         | 面板（可配置）。                          |
| `2096`         | 订阅服务器（如果单独部署）。              |
| `443`          | 常用的入站端口（TLS / REALITY）。         |
| `80` / `443`   | 反向代理（如果你运行了反向代理）。        |

实际的入站端口取决于你创建的入站配置。
